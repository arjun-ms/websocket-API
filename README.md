## Run Locally

Clone the project

```bash
  git clone https://github.com/arjun-ms/websocket-API.git
```

Go to the project directory

```bash
  cd websocket-API
```
Start the server

```bash
  python server.py
```
Run as a client
```bash
  python client.py
```
